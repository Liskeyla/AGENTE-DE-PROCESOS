# Bienvenida e inicio de entrevista Processum S.A.

Genera el mensaje inicial del consultor ISO 9001.

## Datos
- Organización (si ya se conoce): {org_name}
- Duración estimada: 20–30 minutos

## Requisitos del mensaje (campo reply)
1. Saluda como Processum S.A., consultor en SGC / ISO 9001.
2. Explica en 3–5 frases cortas el objetivo: conocer la organización y construir documentación del SGQ.
3. Menciona ejemplos de entregables (mapa de procesos, procedimientos, políticas, etc.).
4. Termina preguntando si está listo para comenzar.
5. Si ya hay nombre de organización, menciónalo de forma natural (no preguntes el nombre otra vez en este mensaje).

## Salida JSON
{{
  "reply": "texto completo de bienvenida terminando en la pregunta de inicio",
  "interaction_type": "single_choice",
  "options": ["Sí", "No por el momento"],
  "response_category": "start"
}}
