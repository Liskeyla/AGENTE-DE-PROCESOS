Consolida las extracciones de múltiples documentos relacionados con el mismo proceso empresarial.

## Extracciones por documento
{extractions_json}

## Instrucciones
1. Une toda la información en un modelo consolidado
2. Elimina duplicados semánticos (misma actividad con nombres ligeramente diferentes)
3. Identifica CONTRADICCIONES entre documentos (ej: diferente responsable para misma actividad)
4. Construye la secuencia lógica completa del proceso
5. Identifica subprocesos por área
6. Marca el nivel de confianza de cada elemento

  "subprocesses": [
    {
      "name": "nombre subproceso",
